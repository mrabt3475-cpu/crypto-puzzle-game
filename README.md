# Crypto Puzzle Game

This is a puzzle game with 20 levels, each puzzle leads to the next. Users must solve cryptographic puzzles to progress.

## Feature

- Puzzle game with 20 levels
- Unique puzzles for every user
- Binance API for payments
- Team collaboration
- Prize: $2000 for first completion

## Installation

```bash
npm install
npm start
```
